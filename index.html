🔥 Connexion Firebase

Pour synchroniser votre collection sur tous vos appareils, connectez un projet Firebase Firestore. C'est gratuit et prend 5 minutes.

1. Allez sur console.firebase.google.com
2. Créez un projet → Firestore Database → mode Production
3. Paramètres du projet → Vos applications → Web → Enregistrer
4. Copiez les valeurs de firebaseConfig ci-dessous
5. Dans Firestore → Règles, collez : allow read, write: if true;

Discogs

Exportez votre collection depuis discogs.com → Collection → Export CSV. Le fichier contient artist, title, label, year, format.

Colonnes attendues :
Artist, Title, Label, Catno, Format,
Rating, Released, release_id
🎧

MusicBee

Dans MusicBee : sélectionnez tous vos morceaux (Ctrl+A) → clic droit → Enregistrer comme playlist → Export M3U. Ou exportez en CSV via Fichier → Bibliothèque → Exporter.

M3U (recommandé) : #EXTINF avec Artist - Title
chemins fichiers : /Artist/Album/track.flac

CSV (alternatif) :
Artist, Album Title, Year, Format, Genre
🔵

last.fm — Connexion API

Connectez directement votre compte last.fm pour récupérer tout votre historique d'écoute automatiquement.

Clé API : last.fm/api/account/create
Nom d'utilisateur : votre pseudo last.fm
Dernière sync : jamais

id, artist, album, year, genre,
cd, flac, mp3, digital, note, plays
