Discogs

Exportez votre collection depuis discogs.com → Collection → Export CSV. Le fichier contient artist, title, label, year, format.

Colonnes attendues :
Artist, Title, Label, Catno, Format,
Rating, Released, release_id
🎧

MusicBee

Dans MusicBee : Fichier → Bibliothèque → Exporter en XML (format iTunes). Ou exportez en M3U/CSV.

XML (recommandé) : Fichier → Bibliothèque → Exporter en XML
Contient artiste, album, année, genre, format, note

M3U (alternatif) : #EXTINF avec Artist - Title
CSV (alternatif) : Artist, Album Title, Year, Format, Genre
🔵

last.fm — Connexion API

Connectez directement votre compte last.fm pour récupérer tout votre historique d'écoute automatiquement.

Clé API : last.fm/api/account/create
Nom d'utilisateur : votre pseudo last.fm
Dernière sync : jamais
